1. Run the 3 Queries in order
2. Use one of the already given logins or simply "Admin", "9999" for all access.
3. Add a new Patient in the "New Patient" form. 
4. Use the Health Number used for New Patient to create a New Admission.
5. Doing so will give you a Admission ID, which you can use in the Patient Billing.
	- Insert the Admission ID into Billing and click "Find"
	- It will give all the needed information along with a total
	- Click on Process to print a reciept, which is also stored in the database
